# Telegram Clock Self Bot
A simple Python self-bot that updates your Telegram last name with the current time.
## Installations
1. clone repository:
```bash
git clone https://github.com/TheToex/clock-bot
```
2. install requirements:
```bash
cd clock-bot
pip install -r requirements.txt
```
## Configuration
Edit the script and enter your Telegram API ID, API Hash, and other required settings before running the bot
get API ID and API Hash from [my.telegram.org](https://my.telegram.org)
## Run
run the bot:
```bash
python clock-bot.py
```
